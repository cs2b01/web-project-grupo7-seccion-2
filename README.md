# Proyecto Final

Se trata de un sitio web de administración de contenido

## Integrantes

## Instalación
Instalar las siguientes extensiones

 Flask

 Flask-SocketIO 

 Flask-Bcrypt

 SQLAlchemy

 Flask-WTF

 Flask-Login

o usar 

```bash
$pip install -r requirements.txt
```

## Descripción
El proyecto usa las metodológias server side y code on demand.
